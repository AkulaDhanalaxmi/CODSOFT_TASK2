# Student Grade Calculator

A console-based Java program that calculates a student's total marks, average percentage, and grade based on marks obtained in multiple subjects.

## Features

- Takes marks obtained (out of 100) for any number of subjects
- Calculates total marks across all subjects
- Calculates average percentage
- Assigns a grade based on the average percentage
- Displays total marks, average percentage, and grade to the user

## How It Works

1. Enter the number of subjects.
2. Enter marks obtained (out of 100) for each subject.
3. The program calculates the total marks and average percentage.
4. A grade is assigned based on the average percentage.
5. Results are displayed: total marks, average percentage, and grade.

## Running the Program

```
javac GradeCalculator.java
java GradeCalculator
```

## Grading Scale

| Average Percentage | Grade |
|---------------------|-------|
| 90 and above         | A+    |
| 80 - 89               | A     |
| 70 - 79               | B     |
| 60 - 69               | C     |
| 50 - 59               | D     |
| Below 50              | F     |
